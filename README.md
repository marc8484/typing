# ⌨️ Typing Hero Challenge
A fun, fast-paced typing game built for my child to practice keyboard skills!

## 🚀 Live Demo
Play the game here: **[https://marc8484.github.io/typing/](https://marc8484.github.io/typing/)**

---

## 🎮 How to Play
1. Words will begin falling from the top of the screen.
2. Type the word exactly as it appears before it hits the bottom.
3. If a word hits the bottom, you lose a life!
4. You have **3 lives** to get the highest score possible.

## 🛠️ Built With
- **HTML5** for the structure.
- **CSS3** for the "Space/Aquarium" dark-mode styling.
- **JavaScript** for the game logic and word spawning.
